# localflow

ローカルタスク管理CLI。プロジェクトのタスクをSQLiteデータベースで管理する。

## グローバルオプション

| オプション | 説明 |
|---|---|
| `--output <FORMAT>` | 出力形式。`text`（デフォルト）または `json` |
| `--project-root <PATH>` | プロジェクトルートディレクトリ。省略時はカレントディレクトリ |

## コマンド

### `localflow add`

新しいタスクを追加する。

### `localflow list`

タスク一覧を表示する。

### `localflow get`

タスクの詳細を表示する。

### `localflow next`

次に着手すべきタスクを表示する。優先度・依存関係を考慮して自動選択する。

### `localflow edit`

既存タスクを編集する。

### `localflow complete`

タスクを完了にする。

### `localflow cancel`

タスクをキャンセルする。

### `localflow deps`

タスクの依存関係を管理する。

### `localflow skill-install`

Claude Code 用の SKILL.md ファイルを生成する。

| オプション | 説明 |
|---|---|
| `--output-dir <PATH>` | 出力先ディレクトリ。省略時はカレントディレクトリ |

## ステータス

| 値 | 説明 |
|---|---|
| `pending` | 未着手 |
| `in_progress` | 作業中 |
| `done` | 完了 |
| `cancelled` | キャンセル済み |

## 優先度

| 値 | 説明 |
|---|---|
| `low` | 低 |
| `medium` | 中（デフォルト） |
| `high` | 高 |
| `critical` | 最優先 |

## 依存関係

タスクは他のタスクへの依存関係を持つことができる。依存先タスクが完了するまで、依存元タスクは `next` コマンドで選出されない。
